The physics of Boom is described in the following papers.

Hydrodynamics

The hydrodynamics of boom uses an implicit Langragian formulation. There is a newtonian and GR scheme which can be controlled by keygr. The Boom code also includes an artificial viscosity scheme. The hydrodynamics also includes a rezoning scheme which was implemented by Joseph Wang based on some ideas with Steve Bruenn.

Neutrino Transport

The neutrino transport uses multi-group flux limited diffusion as described in Bludman et al. 1987). The flux limiter uses a standard Levermore Pomraning formulation.

The reaction rates are taken from Bruenn 1985

Equation of state

The code uses an equation of state by Doug Swesty.

Convection

The convective mechanism in the code was developed by Joseph Wang.
